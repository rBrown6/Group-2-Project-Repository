Group 2's Mission

Hello and welcome to the home page of Group 2! We've dedicated this project to determining which programming language is the most valuable. Within this undertaking, we've come to recognize that the answer to this question is highly subjective and will likely merit a mulitude of diverse responses, each being just as relevant as the next. To many people, terms like "Programming" or "Software Development" simply mean web development. The reality, though, is that many professionals find uses for programming in applications well outside of this scope. Below, we've provided a few examples for the various uses of programming.

Why learn a programming language?

Is your goal as a programmer to create vivid and responsive webpages?
 - Javascript is likely the choice for you.

Are you more interested in interacting with database management systems?
 - You're likely better off learning structured query language.

Does your work require you to maintain antiquated but otherwise irreplaceable software still in use with banks and other financial institutions?
 - It sounds like you may need to have a working knowledge of COBOL.

Is the only reason you're even bothered to learn programming so that you can automate your work load?
 - Maybe you should take a look at Python.

As evidenced by the examples above, we can see that there are several different reasons that someone may opt to employ a programming language, and for very different uses.
