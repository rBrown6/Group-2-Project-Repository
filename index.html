Why learn a programming language?

Is your goal as a programmer to create vivid and responsive webpages?
 - Javascript is likely the choice for you.

 - You're likely better off learning structured query language.

Is the only reason you're even bothered to learn programming so that you can automate your work load?
 - Maybe you should take a look at Python.

As evidenced by the examples above, we can see that there are several different reasons that someone may opt to employ a programming language, and for very different uses.

Ricky's Choice

HTML, while not technically a programming language, is a crucial element to any web-based project and is often the first step of the journey to learning programming. Generally put, HTML is used to build the basic structure of the webpage/application. It works in conjunction with programming languages as well as CSS to provide the structure and flow of your webpage, as well as guide its functionality.

Like stated before, HTML itself does not pass as a programming language on its own. This is because it does not unilaterally perform tasks central to programming, such as creating If/Else statements, declaring variables, writing functions, modifying data or producing output to name a few. Instead, HTML will provide the foundation of the interface used to complete many of these tasks.

Mimi's Favorite Language: SQL

What this language is used for:

  • Database management
  • Sales reports
  • Business management

According to Berkley Extension, SQL, or Structured Query Language, is a language that allows programmers to query and manipulate databases. As a domain-specific language, it is designed mainly for managing data within an RDBMS (relational database management system). Put simply, SQL can locate and retrieve data from a database, as well as update, add or remove records.

While SQL is highly functional, it tends to work better with small databases and doesnâ€™t always lend itself to managing expansive ones.

That said, SQL still ranks as the third-most-used language in the programming industry, with over half (54.7%) of surveyed developers reporting that they use it.

Nichelle's Language Choice: CSS

What this language is used for:

Cascading Style Sheets, fondly referred to as CSS, is a simple design language intended to simplify the process of making web pages presentable.

CSS handles the look and feel part of a web page. Using CSS, you can control the color of the text, the style of fonts, the spacing between paragraphs, how columns are sized and laid out, what background images or colors are used, layout designs,variations in display for different devices and screen sizes as well as a variety of other effects.

CSS is easy to learn and understand but it provides powerful control over the presentation of an HTML document. Most commonly, CSS is combined with the markup languages HTML or XHTML.
